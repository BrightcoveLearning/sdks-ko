    정책 키

    이 항목에서는 계정에 대한 정책 키 생성 및 사용에 대해 알아봅니다.

    서문

    정책 키는 재생 API를 사용하여 Video Cloud 라이브러리에서 비디오 및 재생 목록 콘텐츠를 검색하는 데 사용됩니다. 정책 키는 플레이어에서 설정되므로 플레이어 수준에서 콘텐츠에 대한 액세스를 제한할 수 있습니다. 또한 Video Cloud Studio Media 모듈을 사용하여 비디오수준에서 콘텐츠 가용성을 제어할 수 있습니다.

    브라이트코브 플레이어를 만들면 각 플레이어에 대해 정책 키가 자동으로 생성됩니다. 이 키는 브라이트코브 계정과 연결되어 있으며 비디오가 Video Cloud에 저장된다는 조건으로 볼 수 있는 비디오와 시청할 수 있는 시기를 제어합니다. 예를 들어 다음을 기준으로 액세스를 제한하려면 정책 키가 필요합니다.

    • 계정
    • 도메인 제한 사항 (사용된 경우)
    • 지리적 제한 (사용된 경우)

    정책 키에 대한 자세한 내용은개요: 정책 API문서입니다.

    참고:
    • 정책 키는 Video Cloud에 저장된 비디오를 제어하는 데에만 사용됩니다. 독립형 브라이트코브 플레이어를 사용하는 경우 정책 키에 액세스하거나 검색할 필요가 없습니다.
    • 새로운 브라이트코브 플레이어에 대해 생성된 정책 키는 기본적으로 검색이 활성화되지 않습니다. 재생 API에서 비디오를 검색하려면 정책 API를 사용하여 검색이 활성화된 새 정책 키를 만들어야 합니다. 자세한 내용은정책 API 개요문서를 참조하십시오.

    정책 키 가져오기

    정책 키가 자동으로 생성되고 관리됩니다. 따라서 대부분의 경우 요청하거나 관리 할 필요가 없습니다. 정책 키를 검색하는 가장 쉬운 방법은 다음을 수행하는 것입니다.

    1. 플레이어 모듈을 열고 새 플레이어를 만들거나 플러그인을 추가하려는 플레이어를 찾습니다.
    2. 플레이어의 속성을 열 플레이어 링크를 선택합니다.

    3. 왼쪽 탐색 메뉴에서JSON 편집기를선택하십시오.

    4. 표시된 JSON 코드에서 정책 키를 볼 수 있습니다.

    드문 경우이지만 브라이트코브 플레이어를 만들지 않았지만 정책 키가 필요합니다 (예: 브라이트코브 네이티브 플레이어 SDK를 사용하여 엄격하게 개발하는 경우). 이 경우고객 지원센터에 문의하여 정책 키를 받아야 합니다.

    정책 키 사용

    정책 키를 사용하여 Playback API에 대한 요청을 인증할 수 있습니다. Playback API는 일반 미디어 관리 및 다른 시스템과의 통합을 위해 웹 페이지 또는 모바일 앱에서 런타임에 사용하기 위한 것입니다. 자세한 내용은재생 API인덱스 페이지를 참조하십시오.

    정책 키 대 API 자격 증명

    정책 키는 재생 API에 액세스하는 데 사용되는 반면 API 자격 증명은 다른 Video Cloud API의 콘텐츠에 액세스하는 데 사용됩니다.

    정책 키

    정책 키는 Video Cloud 라이브러리 에셋의 클라이언트 측 검색에 사용됩니다. 자세한 내용은 다음과 같습니다.

    • 재생 API를 사용하여 동영상/재생 목록 데이터에 액세스
    • 읽기 전용
    • 비디오/재생 목록 데이터는 재생 API에 의해 서버에 캐시됩니다.
    • 정책 API를사용하여 다음을 수행합니다.
      • 정책 키를 생성합니다. 정책 키를 취소하려면 고객 지원 센터에 문의해야 합니다.
      • 정책 수준에서 도메인 제한을 설정하고 지리적 제한을 설정합니다. 정책 키를 사용하는 플레이어가 제한을 적용합니다.
    브라이트코브 플레이어의 카탈로그 방법을사용하여 Video Cloud 라이브러리에서 콘텐츠를 검색할 수도 있습니다.

    API 자격 증명

    API 인증 자격 증명 (클라이언트 자격 증명이라고도 함) 을 사용하면 서버 측 애플리케이션이 Video Cloud 라이브러리 자산을 관리할 수 있습니다. 자세한 내용은 다음과 같습니다.

    • 비디오 클라우드 API를 사용하여 데이터에 액세스합니다 . 사용하려는 계정 및 API에 대한 클라이언트 자격 증명이 필요합니다.
    • 읽기, 삽입, 업데이트, 삭제
    • 비디오/재생 목록 데이터가 CMS API에 의해 서버에 캐시되지 않습니다.
    • 비디오 클라우드의 API 인증 UI를사용하여 API 자격 증명 생성 및 관리

    정책 키 관리

    다음과 같이 정책 키를 관리할 수 있습니다.

    얻는 방법

    정책 키를 업데이트할 수 있는 방법은 없습니다. Video Cloud 계정의 기존 플레이어에서정책 키를 가져오거나 Policy API를사용하여 정책 키를 생성할 수 있습니다.

    해지

    정책 키를 취소하려면고객 지원센터에 문의해야 합니다.

    플레이어가 삭제될 때 정책 키는 취소되지 않습니다. 즉, 플레이어의 정책 키를 사용한 다음 플레이어를 삭제해도 계속 정책 키를 사용할 수 있습니다.
