개요

기본적으로 Brightcove Native SDK는 캡션 선택 및 "와 같은 광고 메시지를 포함하여 플레이어 컨트롤에 영어를 사용합니다. 광고를 건너 뛸 수 ", " 더 알아보기 " 그리고 " 광고 1/3 ". 이 문서에 설명된 대로 플레이어 컨트롤을 다른 언어로 현지화할 수 있습니다.

아랍어와 같은 일부 언어의 텍스트는 오른쪽에서 왼쪽(RTL)으로 읽습니다.

기계적 인조 인간

Android용 네이티브 SDK 버전 6.15.1 이상은 다음 언어로 플레이어 현지화를 지원합니다.

  • 아랍어(ar)
  • 영어(ko)

현재 네이티브 SDK는 재생 및 광고 제어를 위해 왼쪽에서 오른쪽(LTR) 레이아웃만 지원합니다.

IMA 광고 작업

IMA 광고로 작업하는 경우 IMA SDK가 텍스트를 자동으로 현지화하지 않는다는 점에 유의하십시오. 따라서 수동으로 설정해야 합니다.

  1. 먼저 장치의 현재 언어를 찾습니다.

    Locale.getDefault().getLanguage();

  2. 그런 다음 초기화할 때 언어를 설정합니다. GoogleIMAComponent :

    ImaSdkSettings settings = ImaSdkFactory.getInstance().createImaSdkSettings();
settings.setLanguage("ar");
googleIMAComponent = new GoogleIMAComponent(
    brightcoveVideoView, 
    eventEmitter, 
    settings);

iOS

iOS용 네이티브 SDK 버전 6.7.4 이상은 다음 언어로 플레이어 현지화를 지원합니다.

  • 아랍어(ar)
  • 중국어, 간체(zh-Hans)
  • 중국어, 번체(zh-Hant)
  • 영어(ko)
  • 프랑스어(fr)
  • 독일어(드)
  • 일본어(자)
  • 한국어(ko)
  • 스페인어

내장된 재생 및 광고 컨트롤은 최신 RTL 레이아웃을 따르며 진행 표시줄은 LTR에서 이동합니다.

IMA 광고 작업

IMA 광고로 작업하는 경우 Brightcove SDK에서 언어를 설정하지 않는다는 점에 유의하십시오. 따라서 수동으로 설정해야 합니다.

다음은 예입니다.

IMASettings *imaSettings = [[IMASettings alloc] init];
imaSettings.language = [NSLocale currentLocale].localeIdentifier;
