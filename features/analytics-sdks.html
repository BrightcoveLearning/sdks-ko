소개

분석 데이터를 사용하면 동영상의 성공을 추적하고 시청자의 습관을 더 잘 이해할 수 있습니다. iOS 및 Android용 Brightcove Native Player SDK를 사용하면 분석 컬렉션을 비디오 솔루션에 쉽게 통합할 수 있습니다.

분석 솔루션:

  • 비디오 클라우드 분석
  • 어도비 애널리틱스(Omniture)

비디오 클라우드 분석

카탈로그 서비스를 사용하는 Video Cloud 고객은 자체 Brightcove Analytics 시스템에서 분석 데이터를 자동으로 수집하게 됩니다. 이 데이터는 집계되어 Video Cloud Analytics 모듈에 표시됩니다. 다음 표는 현재 SDK에서 수집된 데이터 포인트를 강조 표시합니다.

어도비 애널리틱스(Omniture)

콘텐츠 이면의 데이터에 대한 심층 분석을 제공하는 Adobe Analytics는 분석 비즈니스에서 강력한 이름입니다. 이러한 전문 지식을 활용하여 Brightcove Native Player SDK 팀은 iOS 및 Android 플랫폼 모두에서 Adobe Analytics와의 통합을 제공하게 된 것을 기쁘게 생각합니다.

기계적 인조 인간

앱에 Omniture 플러그인을 추가하는 방법에 대한 자세한 내용은 다음 단계를 따르세요.

  1. 시작기본 Omniture 샘플 앱 .

  2. 에서 app/res/layout/omniture_activity_main.xml파일에서 다음을 바꿉니다.

    <com.brightcove.player.view.BrightcoveVideoView

    이것으로

    <com.brightcove.player.view.BrightcoveExoPlayerVideoView

  3. 에서 build.gradle , Omniture 코드 블록을 찾아 다음으로 바꿉니다.

    if (project.name.contains('Omniture')) {
  dependencies {
    implementation "com.brightcove.player:android-omniture-plugin:${anpVersion}"
    implementation files(libraryDirectory + "adobeMobileLibrary.jar")
    implementation "com.brightcove.player:exoplayer2:${anpVersion}"
  }
}

  4. 에서메인 액티비티.자바파일, 찾기onCreate()기능. 어디BrightcoveVideoView생성되면 다음으로 교체하십시오.

    brightcoveVideoView = (BrightcoveExoPlayerVideoView) findViewById(R.id.brightcove_video_view);
  5. ExoPlayer 프레임워크로 샘플 앱을 실행할 준비가 되었습니다.
