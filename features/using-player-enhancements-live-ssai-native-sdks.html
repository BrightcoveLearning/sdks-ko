소개

라이브 SSAI를 위한 Brightcove의 플레이어 향상 기능을 사용하면 광고 수, 광고 카운트다운 타이머, 클릭연결 광고 및 컴패니언 광고를 사용하여 라이브 스트림의 광고 시간을 개선할 수 있습니다. 이러한 라이브 스트림은 Brightcove Native SDK를 사용하여 재생할 수 있습니다.

TV는 웹 브라우저를 지원하지 않으므로 연결된 TV에서는 컴패니언 및 클릭연결 광고를 사용할 수 없습니다.

이 기능을 처음 사용하는 경우 다음을 참조하십시오.

  • 개요: 라이브 SSAI를 위한 플레이어 개선 사항
  • 라이브 SSAI를 위한 플레이어 개선 사항 구현

단계

Brightcove Native SDK를 사용하여 향상된 플레이어로 라이브 스트림을 재생하려면 다음 단계를 따르십시오.

  1. 의 워크플로를 따르십시오. 라이브 SSAI를 위한 플레이어 개선 사항 구현문서, 재생 토큰 생성까지.

  2. 앱 빌드:

    • 안드로이드 구현
    • iOS 구현
  3. 의 워크플로를 계속 진행합니다. 라이브 SSAI를 위한 플레이어 개선 사항 구현문서.

안드로이드 구현

Android용 네이티브 SDK를 사용하여 앱을 빌드하려면 다음 값이 필요합니다.

  • 재생 토큰
  • 계정 ID
  • 동영상 ID
  • 정책 키

정책 키를 받으려면 다음을 검토하십시오. 정책 키문서.

앱을 빌드하려면 다음 단계를 따르세요.

  1. 사용기본 SSAI 샘플 앱코드의 시작점으로 사용할 수 있습니다.
  2. 에서 res/values/strings.xml파일에서 다음을 자신의 값으로 바꿉니다.
    • 계정 ID
    • 동영상 ID
    • 정책 키

  3. 에서메인 액티비티.자바파일, 교체ad_config_id당신의 가치재생 토큰 .

    private final String AD_CONFIG_ID_QUERY_PARAM_VALUE = "your playback token";

  4. NS ad_config_id재생 토큰 값이 HTTP 요청에 쿼리 매개변수로 추가됩니다.

    HttpRequestConfig httpRequestConfig = new HttpRequestConfig.Builder()
  .addQueryParameter(AD_CONFIG_ID_QUERY_PARAM_KEY, AD_CONFIG_ID_QUERY_PARAM_VALUE)
  .build();

  5. 라이브 스트림 비디오 ID와 업데이트된 HTTP 요청을 사용하여 Playback API에 대한 카탈로그 호출을 수행합니다.

    catalog.findVideoByID(getString(R.string.video_id), httpRequestConfig, new VideoListener() {
     @Override
     public void onVideo(Video video) {
         plugin.processVideo(video);
     }
 });
  6. 플레이어가 라이브 스트림을 재생할 준비가 되었습니다.

iOS 구현

iOS용 네이티브 SDK를 사용하여 앱을 빌드하려면 다음 값이 필요합니다.

  • 재생 토큰
  • 계정 ID
  • 동영상 ID
  • 정책 키

정책 키를 받으려면 다음을 검토하십시오. 정책 키문서.

앱을 빌드하려면 다음 단계를 따르세요.

  1. 사용기본 SSAI 플레이어샘플을 코드의 시작점으로 사용하십시오.
  2. 에서 ViewController.swift파일에서 다음을 자신의 값으로 바꿉니다.
    • 계정 ID
    • 동영상 ID
    • 정책 키

  3. 교체AdConfigId당신의 가치재생 토큰 .

    static let AdConfigId = "your playback token"

  4. NS AdConfigId재생 토큰 값이 쿼리 매개변수로 추가됩니다.

    let queryParameters = [kBCOVPlaybackServiceParamaterKeyAdConfigId: Constants.AdConfigId]

  5. 라이브 스트림 비디오 ID 및 쿼리 매개변수를 사용하여 Playback API에 대한 카탈로그 호출을 수행합니다.

    playbackService.findVideo(withVideoID: Constants.VideoId, parameters: queryParameters)
  { [weak self] (video: BCOVVideo?, jsonResponse: [AnyHashable: Any]?, error: Error?) -> Void in
  guard let _video = video else {
    print("ViewController Debug - Error retrieving video: \(error?.localizedDescription ?? "unknown error")")
    return
  }
  self?.playbackController?.setVideos([_video] as NSFastEnumeration)
}
  6. 플레이어가 라이브 스트림을 재생할 준비가 되었습니다.
