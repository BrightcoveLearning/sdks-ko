소개

커넥티드 TV의 사용과 인기는 계속해서 증가하고 있습니다. Brightcove를 사용하면 고화질 콘텐츠가 포함된 대형 스크린 TV로 거실의 청중에게 다가갈 수 있습니다.

Native Player SDK는 사용자 지정 UI 컨트롤, 광고, 콘텐츠 보호(DRM), 라이브 재생, Brightcove 분석 및 QoE 메트릭에 대한 원활한 통합을 제공합니다. 이러한 기능에 대한 자세한 내용은브라이트코브 네이티브 SDK 기능문서.

Brightcove를 사용하면 다음 플랫폼에서 매력적인 TV 앱을 만들 수 있습니다.

  • tvOS용 네이티브 플레이어 SDK
  • Android용 네이티브 플레이어 SDK
  • 나만의 플랫폼

tvOS용 네이티브 플레이어 SDK

tvOS용 Brightcove Native Player SDK를 사용하면 최신 세대의 Apple TV를 위한 매력적인 비디오 경험을 만들 수 있습니다. Native SDK를 사용하면 Video Cloud 라이브러리의 콘텐츠를 사용하여 Apple TV용 비디오 중심 앱을 만들 수 있으며 프리미엄 콘텐츠의 안전한 전달을 위해 FPS(FairPlay Streaming)를 지원합니다.

NS tvOS 샘플 앱이 포함된 기본 SSAI TV 앱의 기반을 제공하여 시작할 수 있습니다.

 

 

이 샘플 앱을 사용하면 동영상 콘텐츠에 광고를 삽입하는 서버 측 광고 연결을 포함하여 재생을 위한 연속 스트림을 제공할 수 있습니다.

   

자세한 내용을 검색하는 개발자라면 다음을 읽어보세요. tvOS용 Brightcove 네이티브 플레이어 SDK기술 문서.

Android용 네이티브 플레이어 SDK

Brightcove 기반 워크플로를 활용하여 청중 범위를 확장하십시오. Android용 Brightcove Native Player SDK를 사용하면 Android 기반 TV 및 Amazon의 Fire TV용 비디오 중심 앱을 만들 수 있습니다.

Android용 기본 SDK를 사용하면 Video Cloud 라이브러리의 고화질 콘텐츠를 활용할 수 있으며 프리미엄 콘텐츠의 안전한 전달을 위해 Widevine Modular를 지원합니다.

NS 기본 Android 샘플 앱 TV 앱의 기반을 제공하여 시작할 수 있습니다.

 

 

앱에 사용자 지정 컨트롤을 도입할 준비가 되면스타일이 지정된 제어 샘플 앱안내를 위해. 이 샘플에서는 설정 아이콘을 추가하여 캡션 표시를 제어하는 방법을 볼 수 있습니다.

 

 

캡션을 켜면 비디오 하단 근처에 캡션이 표시됩니다. 이것은 캡션 파일을 비디오와 연결했다고 가정합니다.

 

 

자세한 내용을 검색하는 개발자라면 다음을 읽어보세요. Android용 Brightcove 네이티브 플레이어 SDK기술 문서.

나만의 플랫폼

원 플랫폼에서 스마트 TV 앱을 개발하려면 Brightcove의 API를 사용하여 Video Cloud 라이브러리와 통합하십시오. 아래 API는 Brightcove Player 및 Brightcove Native SDK에서 사용하는 것과 동일합니다.

재생 API

Playback API를 사용하여 Video Cloud 라이브러리에서 비디오 및 재생 목록 메타데이터를 검색합니다. 이 요청은 현재 재생 URL도 반환합니다. 자세한 내용은 다음을 참조하세요.
