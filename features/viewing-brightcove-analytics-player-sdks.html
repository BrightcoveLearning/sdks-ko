개요

Video Cloud Analytics 모듈에서는 계정 수준 또는 비디오와 같은 세부 수준에서 메트릭 데이터를 볼 수 있습니다. Video Cloud Analytics에서 성능 보고서 보기에 대한 자세한 내용은 다음을 참조하십시오. 실적 보고서 참조문서.

다음은 데이터를 보기 위한 몇 가지 옵션입니다.

  • 계정 ID
  • 목적지
  • 플레이어
  • 원천
Brightcove Analytics는 모든 보고서에서 설정한 대상 및 소스 값을 모두 소문자 문자열 값으로 변환합니다.

당신은 또한 배울 수 있습니다 Video Cloud Analytics에 데이터 보내기 .

계정 ID

앱에서 설정한 소스 값을 봅니다.

accountId - 비디오 수준

accountId를 설정한 비디오 분석을 봅니다.

  1. 분석 모듈에서 다음을 선택합니다. 성능왼쪽 메뉴에서 선택하고동영상나타나는 컨텍스트 메뉴에서

  2. NS 기타/제3자목록 항목에는 accountId 속성을 설정한 모든 동영상의 실적 롤업이 표시됩니다. 현재 Brightcove Analytics는 Brightcove 카탈로그 서비스를 통해 검색되지 않는 비디오에 대한 자세한 보고를 지원하지 않으므로 이러한 비디오는 별도로 나열되지 않습니다.

목적지

앱에서 설정한 대상 값을 봅니다.

대상 - 계정 수준

계정 수준에서 설정한 대상 값을 봅니다.

  1. 분석 모듈에서 다음을 선택합니다. 성능왼쪽 메뉴에서 선택하고디스플레이 도메인나타나는 컨텍스트 메뉴에서

  2. 이 계정 실적 보고서에는 앱 코드에 지정한 값을 포함하여 대상 값이 나열됩니다.

  3. 세부 정보를 보려면 대상 도메인을 선택하십시오.

대상 - 비디오 레벨

비디오 수준에서 설정한 대상 값을 봅니다.

  1. 분석 모듈에서 다음을 선택합니다. 성능왼쪽 메뉴에서 선택하고동영상나타나는 컨텍스트 메뉴에서

  2. 볼 동영상을 선택하세요.

  3. 확장실적드롭다운 메뉴를 선택하고디스플레이 도메인옵션.

  4. 이 동영상 실적 보고서에는 앱 코드에 지정한 값을 포함하여 대상 값이 나열됩니다.

플레이어

앱에서 사용되는 계정 수준의 플레이어를 봅니다.

플레이어 - 계정 수준

앱에서 플레이어에 대한 플레이어 분석을 봅니다.

  1. 분석 모듈에서 다음을 선택합니다. 성능왼쪽 메뉴에서 선택하고플레이어나타나는 컨텍스트 메뉴에서

원천

앱에서 설정한 소스 값을 봅니다.

출처 - 계정 수준

계정 수준에서 설정한 소스 값을 봅니다.

  1. 분석 모듈에서 다음을 선택합니다. 성능왼쪽 메뉴에서 선택하고트래픽 소스나타나는 컨텍스트 메뉴에서

  2. 에서요약목록, 선택추천링크를 클릭하면 출처별 상세 보고서를 볼 수 있습니다.

  3. 이 계정 실적 보고서에는 앱 코드에 지정한 소스 값이 포함됩니다.

소스 - 비디오 레벨

비디오 수준에서 설정한 소스 값을 봅니다.

  1. 분석 모듈에서 다음을 선택합니다. 성능왼쪽 메뉴에서 선택하고동영상나타나는 컨텍스트 메뉴에서

  2. 볼 동영상을 선택하세요.

  3. 확장실적드롭다운 메뉴를 선택하고트래픽 소스옵션.

  4. 이 목록은 앱 코드에서 지정한 소스 값을 포함하여 소스의 성능을 보여줍니다. Analytics 모듈은 다음과 같은 각 목록 항목에 대한 드릴링을 지원하지 않습니다. 직접 .

Video Cloud로 데이터 보내기

앱에서 Video Cloud Analytics로 데이터를 보내는 방법에 대한 자세한 내용은 다음을 참조하십시오.

  • iOS용 브라이트코브 플레이어 SDK
  • Android용 Brightcove 플레이어 SDK

SDK에 포함된 Brightcove 플레이어 및 카탈로그를 사용하여 비디오를 검색하고 재생하면 분석 데이터가 자동으로 수집되고 Video Cloud 분석 모듈에 나타납니다.
