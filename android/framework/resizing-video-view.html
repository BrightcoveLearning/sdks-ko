개요

개발자는 비디오 보기에 비디오가 표시되는 방식에 대한 기본 동작을 변경할 수 있습니다. 내부 비디오 보기( SurfaceView / TextureView ) 채우기 위해BrightcoveVideoView ( FrameLayout ). 이렇게 하려면 다음을 들을 수 있습니다. EventType.VIDEO_SIZE_KNOWN이벤트, 약간의 수학을 수행하고 표시 값을 설정합니다.

비디오 보기를 변경하는 방법에는 두 가지가 있습니다.

  • 컨테이너 크기 조정( BaseVideoView / FrameLayout )
  • 비디오 크기 조정( SurfaceView )

컨테이너 크기 조정

세로 모드에서 전체 장치 화면에 맞도록 비디오 보기를 설정하는 앱을 개발할 수 있습니다. 컨테이너의 크기를 조정하여 이 작업을 수행할 수 있습니다( BaseVideoView / FrameLayout ).

왜냐하면Video높이는 처음에는 알 수 없으며 부모 보기 크기와 일치합니다(전체 화면기본샘플앱 ) 까지VIDEO_SIZE_KNOWN이벤트가 트리거됩니다. 그런 다음, BaseVideoView포장할 것이다SurfaceView . 컨테이너 크기를 조정할 때 초기 깜박임이 나타날 수 있습니다.

컨테이너 속성 설정

비디오 주변에 검은색 여백이 보이면 설정을 시도할 수 있습니다. BrightcoveVideoViewlayout-height .

  1. XML 열기형세파일.
  2. 설정BrightcoveVideoView layout-height에게wrap_content .
  3. 위의 참고 사항에 설명된 대로 초기 깜박임이 발생할 수 있습니다.

비디오 보기 크기 설정

비디오 보기를 제어하는 또 다른 방법은 프로그래밍 방식으로 크기를 설정하는 것입니다.

  1. 초기 깜박임을 최소화하려면layout_height특정 값(예: 280dp). 목표는 높이를 계산된 높이에 가깝게 설정하는 것입니다.
  2. 들어봐EventType.VIDEO_SIZE_KNOWN이벤트.
  3. 가져오기Event.VIDEO_WIDTH그리고Event.VIDEO_HEIGHT
  4. 이 값으로 종횡비를 결정하십시오.
  5. 화면비를 사용하여 디스플레이의 너비와 높이를 계산합니다.

이 솔루션은 다음과 유사합니다.

//Event sent when we know the Video dimensions.
eventEmitter.on(EventType.VIDEO_SIZE_KNOWN, new EventListener() {
  @Override
  public void processEvent(Event event) {
    // Get the width and height
    float width = event.getIntegerProperty(Event.VIDEO_WIDTH);
    float height = event.getIntegerProperty(Event.VIDEO_HEIGHT);
    float aspectRatio = height/width;

    //Get the display metrics
    DisplayMetrics metrics = new DisplayMetrics();
    getWindowManager().getDefaultDisplay().getMetrics(metrics);
    int videoWidth = metrics.widthPixels; // We cover the entire display's width
    int videoHeight = (int) (videoWidth*aspectRatio); //Calculate the height based on the ratio

    // Set the layout params
    brightcoveVideoView.setLayoutParams(new FrameLayout.LayoutParams(videoWidth,videoHeight));
  }
});

보기를 추가할 때 레이아웃 매개변수를 재정의해야 합니다. 자세한 내용은기계적 인조 인간: 보기 추가/레이아웃 관리문서.

비디오 크기 조정

화면을 채우는 두 번 탭을 구현할 수 있습니다. 이것은 16:9 비디오를 촬영하고 18:9 또는 19:9 장치의 화면을 채웁니다.

실제 비디오의 크기를 조정하여 이 작업을 수행할 수 있습니다( SurfaceView ).

  1. 너비와 높이를 설정하면renderView장치 디스플레이의 크기와 일치하도록 비디오가 전체 화면과 일치하도록 크기가 조정됩니다.

    이것은 실제 비디오를 늘리지 않습니다. 다음 단계에서 이를 수행합니다.
    //Event sent when we know the Video dimensions.
eventEmitter.on(EventType.VIDEO_SIZE_KNOWN, new EventListener() {
  @Override
  public void processEvent(Event event) {
    // Get the width and height
    float width = event.getIntegerProperty(Event.VIDEO_WIDTH);
    float height = event.getIntegerProperty(Event.VIDEO_HEIGHT);
    float aspectRatio = height/width;

    //Get the display metrics
    DisplayMetrics metrics = new DisplayMetrics();
    getWindowManager().getDefaultDisplay().getMetrics(metrics);
    int videoWidth = metrics.widthPixels; // We cover the entire display's width
    int videoHeight = (int) (videoWidth*aspectRatio); //Calculate the height based on the ratio

    // Set the video size
    brightcoveVideoView.getRenderView().setVideoSize(videoWidth, videoHeight);
  }
});

  2. 아래 코드는 기본 동영상 보기를 업데이트합니다(예: SurfaceView ) 부모와 일치BaseVideoView ( FrameLayout ). 명심해야 할 몇 가지 사항이 있습니다.

    • 에 대해 들어야 합니다. VIDEO_SIZE_KNOWN전체 화면에 들어가고 나올 때, 그리고 장치를 회전할 때 이벤트.
    • 위의 조건에 대한 보기를 업데이트할 시기를 결정해야 합니다.

    한 가지 시나리오는 다음을 설정할 수 있습니다. BaseVideoView layout_height에게280dp그런 다음 크기를 조정합니다. SurfaceView일치280dp . 가로 세로 비율이 약간 수정됩니다. 사용자가 세로 모드에서 전체 화면으로 이동하고updateVideoViewMatchToParent()호출되면 둘 다BaseVideoView그리고SurfaceView장치의 디스플레이를 채우고 비디오 종횡비가 약간 수정됩니다.

    public void updateVideoViewMatchToParent(@NonNull BaseVideoView brightcoveVideoView) {
  RenderView renderView = brightcoveVideoView.getRenderView();
  if (renderView != null) {
      renderView.setVideoSize(brightcoveVideoView.getWidth(), brightcoveVideoView.getHeight());
  }
}

    더블 탭을 감지하려면 다음을 사용할 수 있습니다. GestureDetector.OnDoubleTapListener상호 작용. 자세한 내용은일반적인 제스처 감지문서.

가로 세로 비율을 고려하지 않고(비디오를 자르지 않고) 비디오를 늘리려면 다음을 시도할 수 있습니다.

eventEmitter.on(EventType.VIDEO_SIZE_KNOWN, new EventListener() {
  @Override
  public void processEvent(Event event) {
      //Get the display metrics
      DisplayMetrics metrics = new DisplayMetrics();
      getWindowManager().getDefaultDisplay().getMetrics(metrics);

      int videoHeight = metrics.heightPixels;
      int videoWidth = metrics.widthPixels;
      brightcoveVideoView.getRenderView().setVideoSize(videoWidth, videoHeight);
  }
});
중요하다BaseVideoView용도match_parent모두layout_width그리고layout_height위해SurfaceView장치의 디스플레이를 채우십시오.

얻는 값은 장치의 방향에 영향을 받습니다. 세로 및 가로로 비디오를 늘리려면 다음을 수행해야 합니다.

  1. 다음 재정의Activity : 

    public void onConfigurationChanged(Configuration configuration)
  2. 비디오 크기를 재설정합니다.
