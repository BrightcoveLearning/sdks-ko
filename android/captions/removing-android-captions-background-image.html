개요

Android용 Brightcove Native Player SDK로 빌드된 앱에 캡션이 있는 비디오를 로드하면 미디어 컨트롤러에 캡션 버튼이 나타납니다.

미디어 컨트롤러에서 캡션 버튼을 선택하여설정단추.

선택설정버튼을 사용하여 구성하는 동안 캡션을 표시하는 동적 배경 이미지를 표시합니다.

자막 배경 이미지 파일의 크기는 400kb입니다. 이 이미지를 제거하기로 선택한 경우 동일한 이름의 빈 이미지 파일로 대체할 수 있습니다.

배경 이미지 제거

기본 캡션 배경 이미지를 제거하려면 다음 단계를 따르세요.

  1. 에서 Android용 브라이트코브 네이티브 SDK , 다음으로 이동합니다. res/drawable-nodpi/caption_background.jpg파일.
  2. 같은 이름의 빈 파일을 만들고, 캡션_배경.jpg .
  3. 기존 교체캡션_배경.jpg같은 이름의 빈 파일이 있는 파일입니다.
  4. 프로젝트를 다시 컴파일하십시오.
