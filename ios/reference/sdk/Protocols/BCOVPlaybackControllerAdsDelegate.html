BCOVPlaybackControllerAdsDelegate 프로토콜 참조

준수 NSObject
선언 BCOVAdvertising.h

개요

각 세션에 대한 기본 광고 정보를 수신하려면 이 프로토콜을 따르십시오.

–  playbackController:playbackSession:didEnterAdSequence:

재생이 재생 세션 내에서 새 광고 시퀀스에 들어갈 때 호출됩니다.

- (void)playbackController:(id<BCOVPlaybackController>)제어 장치 재생세션:(ID<BCOVPlayback세션>)세션 didEnterAdSequence:(BCOVAdSequence *)광고 시퀀스

매개변수

제어 장치

이 전환이 발생한 재생 컨트롤러입니다.
세션

광고 전환이 발생한 재생 세션입니다.
광고 시퀀스

입력 중인 광고 시퀀스입니다.

논의
