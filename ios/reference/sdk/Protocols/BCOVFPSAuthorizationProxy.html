개요

–  contentIdentifierFromLoadingRequest:

선택적 방법. 로드 요청에서 콘텐츠 식별자 검색을 처리하려면 이 메서드를 구현하세요. 이것은 각 자산에 대해 호출됩니다. 이것이 호출되는 스레드에 대해서는 보장되지 않습니다. 이 로딩 요청에 해당하는 콘텐츠 식별자를 반환해야 합니다. 이것은 일반적으로 다음에서 추출됩니다. “ skd:// …” loadingRequest.request.URL의 URL입니다. 콘텐츠 식별자는 SPC를 검색하기 위해 [AVAssetResourceLoadingRequest 스트리밍ContentKeyRequestDataForApp:contentIdentifier:options:error:]에 전달됩니다.

- (nullable NSData *)contentIdentifierFromLoadingRequest:(nonnull AVAssetResourceLoadingRequest *)로딩 요청

이 방법이 제공되지 않으면 기본 방법이 사용됩니다. 기본 방법은 제거“ skd:// ” loadingRequest.request.URL에서 콘텐츠 식별자를 생성합니다.
