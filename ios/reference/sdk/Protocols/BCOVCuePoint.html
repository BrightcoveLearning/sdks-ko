준수 NSObject
선언 BCOVCuePoint.h

개요

Brightcove Player SDK 에코시스템의 큐 포인트를 나타내는 값 클래스. 큐 포인트는 다음을 지정하는 데 사용됩니다. 위치어떤 목적으로든 사용할 수 있는 콘텐츠의 타임라인에 표시 재생이 큐 포인트에 도달하면 재생 세션에서 이벤트를 보냅니다. 큐포인트 BCOVCuePoint 값을 포함하는 신호.

  위치필요한 방법

타임라인에서 큐 포인트가 발생하는 시간 BCOV비디오그것을 소유하는 것.

@property (nonatomic, readonly, assign) CMTime position
