BCOVPlaybackServiceErrorCode 상수 참조

선언 BCOVPlaybackService.h

BCOVPlaybackServiceErrorCode

Playback API 요청과 관련된 오류입니다.

정의

typedef NS_ENUM(NSInteger, BCOVPlaybackServiceErrorCode) {
   BCOVPlaybackServiceErrorNoError,
   BCOVPlaybackServiceErrorCodeConnectionError,
   BCOVPlaybackServiceErrorCodeJSONDeserializationError,
   BCOVPlaybackServiceErrorCodeAPIError,
};

상수

BCOVPlaybackServiceErrorNoError

오류가 없습니다.

선언 BCOVPlaybackService.h .

BCOVPlaybackServiceErrorCodeConnectionError

Playback API에 연결하는 동안 오류가 발생했음을 나타내는 오류 코드입니다. 실제 연결 오류는 error.userInfo 키 NSUnderlyingErrorKey를 확인하십시오.

선언 BCOVPlaybackService.h .

BCOVPlaybackServiceErrorCodeJSONDeserializationError

응답을 JSON으로 구문 분석하는 동안 오류가 발생했음을 나타내는 오류 코드입니다. 요청이 데이터를 반환한 경우 해당 원시 NSData는 kBCOVPlaybackServiceErrorKeyRawResponseData 키에 의해 userInfo에 포함됩니다. 실제 JSON 역직렬화 오류에 대해서는 error.userInfo 키 NSUnderlyingErrorKey를 확인하십시오.

선언 BCOVPlaybackService.h .

BCOVPlaybackServiceErrorCodeAPI오류

API에서 반환된 오류가 있음을 나타내는 오류 코드입니다. API에서 반환된 오류의 NSArray에 대한 error.userInfo 키 kBCOVPlaybackServiceErrorKeyAPIErrors와 반환된 상태 코드를 찾으려면 키 kBCOVPlaybackServiceErrorKeyAPIHTTPStatusCode를 확인하십시오.
