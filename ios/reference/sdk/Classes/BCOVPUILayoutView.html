선언 BCOVPUILayoutView.h

개요

BCOVPUILayoutView 보기 개체는 BCOVPUIBasicControlView . 레이아웃 보기는 일반적으로 BCOVPUIBasicControlView로 생성됩니다. ’ s +layoutViewWithControlFromTag:width:elasticity: 메서드를 포함하고 재생 버튼, 시간 코드 레이블 또는 진행 슬라이더와 같은 표준 플레이어 UI 요소를 포함합니다. 고유한 하위 보기를 추가할 빈 레이아웃 보기를 만들 수도 있습니다. 레이아웃 보기 속성은 가변 너비 레이아웃에서 보기가 서로에 대해 배치되는 방식을 정확히 결정합니다.

  최소폭

레이아웃을 위해 크기를 조정할 때 보기에 설정될 가장 작은 너비입니다. 만약에탄력 0으로 설정하면 뷰의 정확한 너비가 됩니다.

@property(비원자, readwrite) float minimumWidth

  탄력

탄력성은 플레이어 UI의 사용 가능한 너비를 채우기 위해 각 보기의 크기를 조정하는 정도를 결정합니다. ’다른 모든 보기에 상대적인 컨트롤 막대. 컨트롤 막대에 두 개의 레이아웃 보기가 있고 하나는 탄력성이 1.0이고 다른 하나는 탄력성이 2.0인 경우 보기 크기를 조정할 때 후자는 두 배만큼 늘어납니다. 컨트롤 레이아웃의 각 행에는 일반적으로 탄력성이 0보다 큰 레이아웃 보기가 하나 이상 포함되어야 합니다. 탄력성이 0인 보기는 최소 너비로 유지됩니다.

@property(비원자, readwrite) 플로트 탄성
