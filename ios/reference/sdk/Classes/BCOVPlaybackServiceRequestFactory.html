  추가HTTPRequestHeaders

각 NSURLRequest에 설정된 추가 HTTP 요청 헤더의 NSDictionary입니다. 사전 키는 헤더 필드 이름을 정의합니다. 헤더 필드 이름은 대소문자를 구분하지 않습니다. 키와 값은 모두 NSString이어야 합니다. 기본값은 nil입니다.

@property(비원자, 복사, 읽기 쓰기) NSDictionary *additionalHTTPRequestHeaders

선언

BCOVPlaybackServiceRequestFactory.h

–  initWithAccountId:policyKey:

지정된 정책으로 초기화된 인스턴스를 반환합니다. 반환된 인스턴스에서 사용하는 기본 URL 문자열은 다음과 같습니다. https://edge.api.brightcove.com/playback/v1 .

- (인스턴스 유형)initWithAccountId:(NSString *)계정 ID policyKey:(NSString *)정책키

매개변수

계정 ID

비디오 클라우드 계정. 0이 아니어야 합니다.
정책키

Video Cloud CMS 정책. 0이 아니어야 합니다.

반환 값

초기화된 BCOVPlaybackServiceRequestFactory.

논의

지정된 정책으로 초기화된 인스턴스를 반환합니다. 반환된 인스턴스에서 사용하는 기본 URL 문자열은 다음과 같습니다. https://edge.api.brightcove.com/playback/v1 .

Edge Playback Service를 사용하는 경우, 정책키값은 0이어야 합니다.
