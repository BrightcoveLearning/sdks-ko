BCOVIMAAdsRequestPolicy 클래스 참조

다음에서 상속 NSObject
선언 BCOVIMAAdsRequestPolicy.h

개요

지정된 입력 재생 세션에서 사용할 IMAAdsRequests를 생성하기 위한 정책 개체입니다.

+   videoPropertiesVMAPAdTagUrlAdsRequestPolicy

VMAP 광고 태그 URL이 요청할 각 재생 세션의 BCOVVideo 개체를 확인하는 광고 요청 정책을 반환합니다. 비디오 개체의 경우’ s 속성에는 키가 다음과 같은 항목이 있습니다. kBCOVIMAAdTag , 해당 항목의 값은 VMAP 광고 태그 URL을 포함하는 NSString으로 간주됩니다. 해당 URL은 해당 재생 세션에 대한 VMAP IMAAdsRequest를 구성하는 데 사용됩니다.

+ (인스턴스 유형)videoPropertiesVMAPAdTagUrlAdsRequestPolicy

반환 값

각 재생 세션의 정보에서 VMAP IMAAdsRequests를 생성하는 광고 요청 정책’ NS BCOVVideo.properties .

논의

VMAP 광고 태그 URL이 요청할 각 재생 세션의 BCOVVideo 개체를 확인하는 광고 요청 정책을 반환합니다. 비디오 개체의 경우’ s 속성에는 키가 다음과 같은 항목이 있습니다. kBCOVIMAAdTag , 해당 항목의 값은 VMAP 광고 태그 URL을 포함하는 NSString으로 간주됩니다. 해당 URL은 해당 재생 세션에 대한 VMAP IMAAdsRequest를 구성하는 데 사용됩니다.

다른 비디오에 대해 다른 VMAP 광고 태그 URL을 지정하는 기능을 원할 때 이 정책을 사용하십시오.

선언

BCOVIMAAdsRequestPolicy.h

+   adsRequestPolicyWithVMAPAdTagUrl:

모든 재생 세션에 대해 지정된 광고 태그 URL 및 컴패니언 슬롯을 사용하여 VMAP IMAAdsRequest를 생성하는 광고 요청 정책을 반환합니다.

+ (인스턴스 유형)adsRequestPolicyWithVMAPAdTagUrl:(NSString *)VMAPAdTagUrl

매개변수

VMAPAdTagUrl

이 정책이 생성하는 IMAAdsRequest에 포함할 광고 태그 URL입니다. 현재는 VMAP 광고 태그 URL만 지원됩니다.

반환 값

지정된 매개변수에서 VMAP IMAAdsRequests를 생성하는 광고 요청 정책입니다.

논의

모든 재생 세션에 대해 지정된 광고 태그 URL 및 컴패니언 슬롯을 사용하여 VMAP IMAAdsRequest를 생성하는 광고 요청 정책을 반환합니다.

모든 동영상에 동일한 VMAP 광고 태그 URL을 사용하려는 경우 이 정책을 사용합니다.

선언

BCOVIMAAdsRequestPolicy.h

+   adsRequestPolicyWithVASTAdTagsInCuePointsAndAdsCuePointProgressPolicy:

유형의 BCOVCuePoint에 대해 각 BCOVVideo를 확인하는 광고 요청 정책을 반환합니다. ‘ kBCOVIMACuePointTypeAd ’ . 큐포인트 개체가’ s 속성에는 키가 다음과 같은 항목이 있습니다. kBCOVIMAAdTag , 해당 항목의 값은 VAST 광고 태그 URL을 포함하는 NSString으로 간주됩니다. 해당 URL은 해당 재생 세션에 대한 VAST IMAAdsRequest를 구성하는 데 사용됩니다.

+ (인스턴스 유형)adsRequestPolicyWithVASTAdTagsInCuePointsAndAdsCuePointProgressPolicy:(BCOVCuePointProgressPolicy *)adsCuePointProgressPolicy

매개변수

adsCuePointProgressPolicy

VAST 광고에 대해 지정된 큐 포인트 진행 정책은 큐 포인트를 요청합니다. 이 매개변수가 nil이면, +[BCOVCuePointProgressPolicy progressPolicyProcessingCuePoints:BCOVProgressPolicyProcessAllCuePoints resumingPlaybackFrom:BCOVProgressPolicyResumeFromContentPlayhead 무시PreviouslyProcessedCuePoints:NO]기본값으로 사용됩니다.

반환 값

각 재생 동영상의 정보에서 VAST IMAAdsRequests를 생성하는 광고 요청 정책’ NS BCOVCuePoint.properties .

논의

유형의 BCOVCuePoint에 대해 각 BCOVVideo를 확인하는 광고 요청 정책을 반환합니다. ‘ kBCOVIMACuePointTypeAd ’ . 큐포인트 개체가’ s 속성에는 키가 다음과 같은 항목이 있습니다. kBCOVIMAAdTag , 해당 항목의 값은 VAST 광고 태그 URL을 포함하는 NSString으로 간주됩니다. 해당 URL은 해당 재생 세션에 대한 VAST IMAAdsRequest를 구성하는 데 사용됩니다.

다른 큐포인트에 대해 다른 VAST 광고 태그 URL을 지정하는 기능을 원할 때 이 정책을 사용하십시오.
