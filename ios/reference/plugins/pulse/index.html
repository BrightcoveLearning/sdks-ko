설치

Brightcove Player SDK용 Pulse 플러그인은 설치를 위한 동적 라이브러리 프레임워크를 제공합니다.

Pulse 플러그인은 iOS용 INVIDI Technologies Pulse SDK 버전 2.5.20.1.0 및 tvOS용 버전 2.5.19.9.0을 지원합니다. Pulse iOS 및 tvOS SDK 참조 . Brightcove Player SDK 참조용 Pulse 플러그인에 대한 Podspec Pulse SDK 출시 내역 .

재생 및 일시 중지

Brightcove Pulse 플러그인은 사용자 지정 재생 및 일시 중지 논리를 구현하여 가능한 한 가장 부드러운 광고 경험을 보장합니다. 따라서 재생 방법을 사용하고 있는지 확인해야 합니다. BCOV재생 컨트롤러아니면 그 [BCOVSessionProviderExtension pulse_play]또는 [BCOVSessionProviderExtension pulse_pause] ( BCOVSessionProvider 확장 ), AVPlayer가 아닙니다.

예를 들어 처음으로 play를 호출하는 경우 BCOV재생 컨트롤러 BCOVPulse는 프리롤 전에 재생되는 콘텐츠 없이 프리롤 광고를 처리할 수 있습니다. BCOVPulse가 기본값을 재정의하는 방법에 대한 자세한 내용은 BCOV재생 컨트롤러방법을 확인하십시오 BCOVSessionProvider 확장 .

플러그인 동작 사용자 정의

BCOVPulse에는 몇 가지 구성 지점이 있습니다. BCOVPulse를 iOS용 Brightcove Player SDK용 다른 플러그인과 결합하고 사용자 정의 보기 전략을 생성하고 현재 광고 요청을 재정의할 수 있습니다.

BCOVPulsePlaybackSessionDelegate

NS BCOVPulsePlaybackSessionDelegate프로토콜은 BrightcovePulse 플러그인이 호스트 응용 프로그램과 통신할 수 있는 방법을 제공합니다. 이 대리자를 사용하면 세션이 시작되기 전에 콘텐츠 메타데이터 및 요청 설정을 재정의할 수 있습니다.

NS UIViewController채택할 필요가 있다 BCOVPulsePlaybackSessionDelegate규약.

@interface ViewController() <BCOVPulsePlaybackSessionDelegate>

전달하는 NSDictionary 생성 kBCOVPulseOptionPulsePlaybackSessionDelegateKey세션을 재정의하는 메서드를 구현할 클래스와 함께.

NSDictionary *펄스옵션 = @ { kBCOVPulseOptionPulsePlaybackSessionDelegateKey: self } ;

만들기 BCOVPlaybackSessionProvider또는 BCOV재생 컨트롤러의 방법을 사용하는 Pulse의 경우 BCOVPlaybackManager , 이전 단계에서 만든 사전을 전달하는 것을 잊지 마십시오. 이 예에서는 재생 컨트롤러를 사용합니다.

BCOVPlayerSDKManager *관리자 = [BCOVPlayerSDKManager 공유 관리자]; id<BCOVPlaybackController> 컨트롤러 = [관리자 createPulsePlaybackControllerWithPulseHost:pulseHost contentMetadata:[OOContentMetadata new] requestSettings:[OORequestSettings new] adContainer:self.playerView.contentOverlayView 동반자 슬롯:nil viewStrategy:nil 옵션:pulseOptions];

NS createSessionForVideo:withPulseHost:contentMetadata:requestSettings:메소드는 현재 비디오, 호스트 도메인, 콘텐츠 메타데이터그리고요청 설정세션을 위해. 이 예에서 이전 개체는 비어 있었고 새 개체로 재정의됩니다. 범주그리고선형 재생 위치정렬.

- (id<OOPulseSession>)createSessionForVideo:(BCOVVideo *)video withPulseHost:(NSString *)pulseHost contentMetadata:(OOContentMetadata *)contentMetadata requestSettings:(OORequestSettings *)requestSettings { // Override the content metadata. contentMetadata.category = @"new_category_for_midrolls"; // Override the request settings. requestSettings.linearPlaybackPositions = @[ @(15) ]; return [OOPulse sessionWithContentMetadata:contentMetadata requestSettings:requestSettings]; }
