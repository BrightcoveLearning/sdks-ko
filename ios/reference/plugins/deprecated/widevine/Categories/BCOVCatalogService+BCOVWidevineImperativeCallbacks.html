선언 BCOV카탈로그서비스+BCOVWidevineAdditions.h

개요

이러한 범주 메서드에 추가된 파인더 메서드의 기능은 결과(또는 오류)를 전달하는 신호를 반환하는 대신 이러한 메서드가 void를 반환하지만 결과(또는 오류). ReactiveCocoa를 사용하는 것이 편하다면’ s 신호 API인 BCOVatalogService에서 제공하는 신호 방법은 더 나은 유연성을 제공합니다. 이러한 범주 메서드는 신호를 직접 사용하지 않으려는 클라이언트를 위한 것입니다.

–  findWidevinePlaylistWithPlaylistID:매개변수:완료:

재생 목록 ID로 Media API 서비스에서 BCOVPlaylist를 검색합니다.

- (void)findWidevinePlaylistWithPlaylistID:(NSString *)재생 목록 ID 매개변수:(NSDictionary *)매개변수 완료:( 무효 ( ^ ) ( BCOVPlaylist *playlist , NSDictionary *jsonResponse , NSError *error ))완료 핸들러

매개변수

재생 목록 ID

찾을 재생 목록의 ID입니다.
매개변수

Media API 요청에 추가할 추가 NSString 쿼리 매개변수입니다. 이러한 값은 충돌하는 경우 기본값을 재정의합니다.
완료 핸들러

요청이 완료되면 호출될 블록입니다. completionHandler의 실행은 메인 스레드에서 발생합니다.
