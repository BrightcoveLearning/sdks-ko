    브라이트코브 네이티브 SDK

    브라이트코브 네이티브 SDK 기능 수준 설명서에 오신 것을 환영합니다! 여기에서 모바일 및 TV 장치용 기본 SDK에 대해 알아야 할 모든 것을 찾을 수 있습니다.

    개요

    네이티브 SDK는 비디오 플레이어 자체가 아닙니다. 대신 광고, 분석 및 디지털 권한 관리 및 폐쇄 자막과의 통합 지점으로 볼 수 있습니다.

    네이티브 SDK에 대한 간단한 요약을 보려면 다음 비디오 소개를 확인하십시오.

     

     

    네이티브 개발은 복잡할 수 있으며 여러 가지 방법으로 응용 프로그램에 비디오를 가져올 수 있습니다. 주제를 처음 사용하는 경우 기본 환경에서 비디오 재생 옵션을 읽는 것이 좋습니다.

    브라이트코브 네이티브 SDK를 사용하는 이유는 무엇입니까? 읽기네이티브 비디오 재생안내서.

    기본 SDK를 사용하여 간단한 비디오 앱을 만드는 방법을 알아보려면 다음 동영상을 확인하세요.

    •  비디오 교육 : 네이티브 SDK로 개발

    특징

    브라이트코브 SDK는 다음과 같은 다양한 기능을 지원합니다.

    • 자막
    • 광고
    • 분석 도구
    • 콘텐츠 보호
    • 라이브 재생
    • 표준 UI 컨트롤
    • 360° 비디오
    • 오프라인 재생

    전체 기능 목록은 Brightcove 기본 SDK 기능문서.

    기능 수준 정보에 대한 최신 정보 유지 Brightcove 블로그 .

    제품

    네이티브 플랫폼 개발자를 염두에 두고 설계된 브라이트코브 네이티브 SDK는 각 환경에서 애플리케이션 개발의 최신 표준을 준수하여 출시 기간을 최소화하는 것을 목표로 합니다.

    브라이트코브 네이티브 SDK는 네이티브 플레이어 프레임워크를 기반으로 구축됩니다. 이를 통해 네이티브 OS의 속도, 성능 및 유연성을 활용하고 통합하기 쉬운 완벽한 비디오 환경으로 확장할 수 있습니다. 또한 네이티브 SDK는 메모리, 이름 지정 및 디자인 패턴에 대해 설정된 규칙을 따라 최대한 간단하게 시작할 수 있습니다.

    기본 SDK는 다음 유형의 디바이스와 통합되도록 설계되었습니다.

    • 모바일 장치
    • 연결된 TV / 셋톱 박스

    모바일 기기

    당사는 모바일 장치에 대해 다음과 같은 SDK를 제공하며, 각 SDK는 각자의 특정 목적을 위해 전문화되어 있습니다.

    • iOS 용 Brightcove 네이티브 SDK
    • Android 용 Brightcove 기본 SDK

    iOS용 브라이트코브 네이티브 SDK

    iOS용 브라이트코브 네이티브 SDK는 최신 iOS 규칙을 따르기 때문에 기존 Objective-C 앱이나 새로운 Swift 앱에 통합하는 경우 집에서도 안심할 수 있습니다.

    지원되는 iOS 버전
    Apple의 최신 기술 개선 사항을 활용하기 위해 현재 다음과 같은 지원을 제공합니다.

    • 아이폰 OS
    • 아이폰 OS
    • 아이폰 OS

    iOS 9 및 Xcode 7의 출시와 함께 새로운 기능으로 작업하는 방법을 배우십시오. 앱 전송 보안 (ATS) .

    iOS용 브라이트코브 SDK는 내부적으로 수집된 지표와 Apple의 공개 iOS 기기 사용 데이터 보고서를 결합하여 대다수의 사용자에게 도달하는 것을 목표로 하고 있습니다.

    자세한 내용을 검색하는 개발자 인 경우 iOS 용 Brightcove 네이티브 SDK기술 문서.

    브라이트코브 네이티브 SDK

    Android용 브라이트코브 네이티브 SDK는 최신 버전의 Android에서 출시된 개선 사항을 활용하는 동시에 대다수의 사용자에게 도달하는 균형을 유지하는 것을 목표로 하고 있습니다.

    지원되는 Android 버전
    이러한 도달 범위와 Google에서 출시 한 개선 사항을 활용할 수있는 기능의 균형을 맞추기 위해 현재 다음 Android 버전에 대한 지원을 제공합니다.

    • 10.0
    • 9.0
    • 8.0, 8.1
    • 7.0, 7.1
    • 6.0

    내부적으로 수집 된 측정 항목과 Google의 공개를 조합하여 사용 Android 기기 사용 데이터보고서에 따르면 Brightcove SDK for Android는 대다수의 사용자에게 도달하는 것을 목표로합니다.

    자세한 내용을 검색하는 개발자라면Android용 브라이트코브 네이티브 SDK기술 문서를 참조하십시오.
