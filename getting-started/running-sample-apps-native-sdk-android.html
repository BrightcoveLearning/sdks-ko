    안드로이드용 네이티브 SDK 용 샘플 앱 실행

    이 주제에서는 Android용 브라이트코브 네이티브 SDK와 연결된 샘플 앱을 실행하는 방법에 대해 알아봅니다. 이러한 샘플은 고유한 앱을 만들기 위한 좋은 출발점을 제공합니다.

    개요

    Android용 브라이트코브 네이티브 SDK는 자체 구현을 시작하는 데 도움이 되는 다양한 샘플 앱을 제공합니다. 전체 샘플 목록은 다음을 참조하십시오.

    • Android 용 플레이어 샘플
    샘플 앱 외에도 유용한 코드 조각 모음도 있습니다. 그들은에서 찾을 수 있습니다 Android 용 네이티브 SDK를 사용하는 코드 스 니펫문서.

    브라이트코브 네이티브 SDK를 처음 사용하는 경우기본 SDK 시작하기비디오를 확인하십시오.

    프레임워크

    Android 샘플 앱은 최신 버전의 브라이트코브 네이티브 SDK 및 플러그인을 지원합니다.

    네이티브 SDK에서 사용할 수 있는 최신 기능을 구현하려면 앱에 ExopLayer 2 프레임워크를 사용하는 것이 좋습니다. 현재이 프레임 워크를 사용하지 않는 경우 ExoPlayer 2 프레임 워크로 마이그레이션문서.

    샘플 앱 설치

    Android 샘플 앱은 최신 버전의 브라이트코브 네이티브 SDK 및 플러그인을 지원합니다.

    현재 모든 Android 샘플 앱은 번들로 함께 설치해야 합니다. 시작하는 가장 쉬운 방법은 샘플 앱을 Android Studio로 가져오는 것입니다. 이것이 우리가 다음 단계에서 수행 할 것입니다.

    샘플에 대한 자세한 내용과 설치를위한 버전 제어 옵션에 대한 자세한 내용은 Android 플레이어 샘플 추가 정보파일.

    샘플 앱을 가져오려면 다음과 같이 하십시오.

    1. 아직 가지고 있지 않다면 최신 버전을 다운로드하여 설치하십시오. 안드로이드 스튜디오 .
    2. 복제 Android 용 플레이어 샘플컴퓨터에.
    3. Android Studio에서프로젝트 가져 오기 .

    4. 파일 선택 대화 상자에서 최상위 레벨을 선택하십시오. build.gradle로컬 복제 저장소의 파일.

      Android 샘플 앱용 네이티브 SDK는 버전 3.1.4까지 Gradle 빌드 프로세스를 사용하여 테스트되었습니다.
    5. Android Studio에서는 모든 샘플 앱 프로젝트를 볼 수 있습니다.

    샘플 앱 실행

    샘플 앱은 다음을 사용하여 실행할 수 있습니다.

    • 안드로이드 스튜디오 에뮬레이터
    • 컴퓨터에 연결된 물리적 장치를 사용하는 안드로이드 스튜디오
      • 기기에 안드로이드 OS 4.4.4 이상이 있어야 합니다.
      • 장치에서 USB 디버깅이 활성화되어 있는지 확인하십시오.

    샘플 앱을 실행하려면 다음과 같이 하십시오.

    1. Android Studio의 왼쪽 프로젝트 목록에서 BasicSampleApp계획.
    2. 확장자바폴더를 열고 MainActivity.java파일.
    3. 프로젝트 선택 드롭 다운에서 BasicSampleApp .
    4. 프로젝트 선택 드롭 다운 오른쪽에서운영단추.
    5. 컴퓨터에 연결된 장치가있는 경우 해당 장치가배포 대상 선택대화. 그렇지 않으면 가상 디바이스를 만든 후 여기에 나열된 가상 디바이스도 표시됩니다. 장치를 선택하고확인 .
    6. 장치 또는 에뮬레이터에서 실행되는 앱이 표시되어야합니다.

    디버깅

    Android Studio에는 중단 점을 설정하고 변수를 검사하고 런타임에 표현식을 평가할 수있는 디버거가 있습니다. 에뮬레이터에서 앱을 디버깅하려면 다음을 수행하십시오.

    1. Android Studio에서 실행을 일시 중지하려는 코드 줄을 찾습니다.
    2. 코드 줄의 왼쪽 거터에서 을 클릭하여 중단점을 추가합니다.
    3. 선택디버그버튼을 눌러 실행을 시작합니다.

    4. 중단 점에서 실행이 일시 중지되면 변수 및 표현식을 검사하는 몇 가지 옵션과 코드 줄을 단계별로 통과하거나 넘어가는 옵션이 있습니다.

      앱 디버깅에 대한 자세한 내용은디버깅을위한 Android Studio의 사용자 가이드 .
